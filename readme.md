API REST – Gestión de Usuarios y Tareas

Proyecto construido con Node.js, Express, MySQL,  validaciones con Zod, middlewares personalizados y control de acceso basado en roles.

Tecnologías Utilizadas

Node.js 22

Express.js

MySQL2

Dotenv

Nodemon

Zod (validaciones)

ES Modules


Instalar dependencias:

npm i


Modo desarrollo:

npm run dev


tambien considerar el usar el contenido del archivo database.sql, creando una base de datos primero y usando el archivo para crear las tablas.

llenar los campos del archivo .env.example y renombrarlo a .env
